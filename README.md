# Ned's TIL

Source for [nedjwestern.github.io](https://nedjwestern.github.io/), a personal "Today I Learned" site built with [Jekyll](https://pages.github.com/) and the [Just the Docs](https://github.com/just-the-docs/just-the-docs) theme (via `remote_theme`). Short, practical tutorials on Podman, Python, and R, written the way I'd have wanted to read them when I was first learning the material.

Tutorial pages live under `tutorials/`. See `CLAUDE.md` for the site's structure and content conventions.
